# 🛒 Funnel Analysis – E-commerce Dataset (2019)

## 📌 Overview
This project performs a **funnel analysis** on a 2019 e-commerce cosmetics dataset to understand user behavior across key stages of the purchase journey.

The goal is to identify how users move through the funnel and where drop-offs occur:
- Product views  
- Add to cart  
- Purchase  

---

## 🎯 Objectives
- Analyze user progression through the sales funnel  
- Calculate conversion rates between stages  
- Identify where users drop off  


---

## 📊 Funnel Stages
1. **View** – User views a product  
2. **Cart** – User adds product to cart  
3. **Purchase** – User completes a purchase  

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- Matplotlib  
- NumPy  

---

## 📈 Key Metrics
- Number of users at each stage  
- View → Cart conversion rate  
- Cart → Purchase conversion rate  

---

## 📉 Example Insights
- Identify stages with the highest drop-off  
- Evaluate potential improvements in user experience  
- Compare conversion performance between groups
